# Python Backend API

This project is a simple backend REST API built using Python and Flask.
It demonstrates core backend development concepts and REST API fundamentals.

## Technologies Used
- Python
- Flask
- MySQL (database design reference)

## Features
- Create and retrieve users
- REST API structure
- Clean backend code organization

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the application:
   python app.py

This project was built to strengthen backend development skills.
